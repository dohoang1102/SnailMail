UNDER-Ware

("User Now Defines Entity Rights" software licensing)

The UNDER-Ware software license agreement is designed to be one of the most user-friendly software licenses ever.  The driving principle behind this license agreement is that the USER gets to choose at least some aspects of the license UNDER which they install, use and distribute the licensed entity from a list of options included with the license (see blow) itself.  Some people are not comfortable with "One size fits all" products, hence the UNDER-ware motto is:

Your UNDER-Ware should be comfortable.
Note that which ever software license options the software is used UNDER, there are some additional license requirements that the software is bound by:
 
  • The software must not be distributed without the complete package including the license agreement, documentation, etc.
  • The software must not be used for any commercial purposes without the written permission of the author.
  • The software remains the copyright property of the author.

UNDER-Ware License Options

Here is a list of options from which the UNDER-Ware license holder may choose:
  • Shareware - if you like the product and wish to keep using it, please send some money to the author(s) as they've put a lot of effort into it, and like to know that it's been appreciated.  Whatever you think is a fair amount will do.
  • Beerware - similar to shareware, but instead of money, send some of your local brew (surface mail only - beer doesn't much like the lack of air-pressure at high altitudes).
  • Haggleware - Don't like any of the other options?  Got some other way to show your appreciation?  Let the developer(s) know.  They may be happy to accept other options you come up with.
